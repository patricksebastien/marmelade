# Screenshots

Drop the PNG/JPG files referenced by the top-level README.md into this folder:

- `overview.png`   — full app window (hero shot)
- `view-modes.png`  — waveform render/view modes (spectrogram etc.)
- `mastering.png`   — the per-keeper Mastering chain dock
- `markers.png`     — the Keepers + Markers side panels

Recommended width ~1600px, PNG. The README sizes them responsively.
